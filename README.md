# Código Neutrón - Landing Page

Landing page profesional para el método **Código Neutrón**, un proceso de intervención sobre patrones mentales y gestión de energía para profesionales de alto rendimiento, creado por **Patricia Pietrasanta**.

## 🚀 Tecnologías

- **React 19**
- **Vite** (Build tool)
- **Tailwind CSS 4** (Styling)
- **Motion/React** (Animaciones)
- **Lucide React** (Iconografía)

## 🎨 Arquitectura y Diseño

El proyecto ha sido rediseñado bajo una arquitectura **modular** para facilitar el mantenimiento y la escalabilidad. Cada sección de la landing vive en su propio componente dentro de `src/components/`.

### Estética Premium
- Tipografía Serif (`Instrument Serif`) para títulos que evocan autoridad y madurez.
- Paleta de colores neutra (`#F8F7F4`, `#1A1A1A`) con acentos dorados (`#B49F60`).
- Textura de grano editorial para un acabado orgánico y de alta gama.
- Enfoque **Mobile-First** con una experiencia de usuario optimizada para dispositivos táctiles.

## 🛠️ Instalación y Desarrollo

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Construir para producción:
   ```bash
   npm run build
   ```

## 📋 Estructura de Componentes

- `Hero`: Propuesta de valor inicial y CTA principal.
- `PainPoints`: Autodiagnóstico de fricciones comunes.
- `EnergyAudit`: Herramienta interactiva para evaluar áreas de vida.
- `HowItWorks`: Explicación de la metodología.
- `ProcessSection`: Fases del proceso individual.
- `Testimonials`: Prueba social de profesionales reales.
- `Bio`: Perfil profesional de Patricia Pietrasanta.
- `Investment`: Detalles de la inversión y garantía.
- `FAQ`: Preguntas frecuentes resueltas.
